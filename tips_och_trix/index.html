3 Wamp Server

Wamp Server är ett utomordentligt verktyg att sätta upp en server hemma på sin egen dator med tillgång till Apache Tomcat, MyPHPAdmin, och webhost. Allt placerat på servern publicerar såklart lokalt och inte publikt.

Det finns ett par problem som många nya användare stöter på direkt vid första användningen av WAMP Server och det är att servern inte går online, och att man inte vet vart man ska placera sina saker. Här nedan tänker jag reda ut hur man går tillväga på en dator med Windows som operativsystem. Använder du ett annat operativ så borde det vara ungefär lika, men om det inte är det, googla.

Om du stöter på problemet med att din server inte går online, så är det mest troligt på grund utav att du har Skype installerat och att Skype ligger och pratar på samma portnummer som WAMP. Gör följande steg:
